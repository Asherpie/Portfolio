Magma Gems
With this Unreal project, my job was to create the crystal substance and material using Substance and the Unreal Engine. I used substance to create the base textures for surface texturing and plugged those into the Unreal material node tree in order to create a parent crystal material with various inputs for changing color, emission, magma crust, etc. to individualize each crystal instance.
